This package contains the components used to design the GUI of game.

The font used is "AG Book Rounded".
Text fields are implemented you just have to assign the font in the inspector view. 

There are eight scenes provided each is having multiple dialogs that can be used in different scenes.
prefabs are given for every dialog that has been used in every scene.
simple insert the prefabs to canvas to insert the predefined dialogs.
you can create your own dialogs by adding buttons and other fields and applying the sprites to them.
You can ad the sprite to any field inside the "Image" property under "Inspector" window

All the animations are applied using the transition property of buttons
